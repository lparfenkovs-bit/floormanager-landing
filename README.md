# floormanager-landing

Multi-page marketing site for [floormanager.app](https://floormanager.app).

## Pages

- `index.html` - homepage and top-level value proposition
- `features.html` - product and workflow detail
- `pricing.html` - sales-ready offer copy
- `contact.html` - sales contact and lead capture guidance

## Shared files

- `styles.css` - visual design and responsive layout
- `app.js` - mobile navigation, scroll reveal, and active-page handling
- `PUBLISH.md` - GitHub sync prep and Windows-safe publish flow

## Local preview

Open `index.html` directly in a browser or serve the folder with any static file server.
