Operational detail

Built for the rules you cannot break.

Each feature reflects a floor constraint, not a generic dashboard idea. That keeps the interface small, readable, and enforceable.

01

Saw gating

Downstream work cannot start until Saw is marked finished with a saved cut quantity.

02

Sequential routing

Jobs move through the route in strict order, with Quality Check and Packaging always last.

03

Operator timing

Timers and workflow changes stay behind explicit permissions so operators only see the controls they need.

04

Good and reject counts

Counts become editable after the job starts, keeping production state aligned with the live floor.

05

Inventory automation

Stock deduction happens automatically when a work order completes, regardless of inventory permissions.

06

Search that understands dimensions

Material search supports formats like 10x30 and 20 mm, so operators can find stock quickly.

Floor rule

No parallel Saw, no loose ends.

The route does not fork around the Saw station. Once the cut is saved, the next workcenter unlocks, and the line continues toward final quality check and packaging.

Why this matters

Less ambiguity on the floor, fewer surprises in the system.
