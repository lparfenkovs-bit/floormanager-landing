Platform capabilities

Feature blocks that make the system feel built for production, not presentation.

01

Saw gating

Downstream stations cannot start until Saw is finished and the cut quantity has been saved.

02

Strict routing

The route executes in order from first station to the final closeout steps.

03

Total-time closeout

Quality Check and Packaging use total-time behavior instead of the usual setup and machining split.

04

Time input permissions

Operators need explicit permission to trigger timers and move workflow states.

05

Production-ready counts

Good and reject counts stay locked until the job starts, which keeps live state readable.

06

Dimension-aware search

Search understands material formats like 10x30 and 20 mm to speed up floor lookup.
