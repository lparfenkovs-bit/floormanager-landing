FAQ

Clear answers before the first live run.

Can a team skip ahead?

No. The workflow follows a fixed sequence so the next step only opens when the current one is done.

Do users need admin access?

No. Settings stay admin-only while regular users get only the permissions they need.

How are updates handled?

Status changes happen automatically when the workflow is completed.
